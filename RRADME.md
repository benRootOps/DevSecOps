DevSecOps 56 Days Challenge

**Objectif : Devenir Junior DevSecOps Engineer en 8 semaines | 4h/j, 7j/7**

Ce dépôt trace mon parcours intensif, jour après jour, avec des livrables concrets. 
L’idée : Passer de zéro à "embauchable" avec un portfolio AWS + Sécurité + CI/CD.

**Auteur** : MBA Joseph Benjamin  
**Localisation** : Yaoundé, Cameroun  
**Stack Cible** : Kali Linux, AWS, Docker, Kubernetes, Terraform, GitHub Actions

---

🎯 Objectifs du Challenge

À la fin des 56 jours, je serai capable de :
1.  **Provisionner une infra sécurisée** sur AWS avec Terraform + CIS Benchmarks
2.  **Construire un pipeline CI/CD** avec scans SAST, SCA, et gestion des secrets
3.  **Déployer et sécuriser** des applications sur Kubernetes EKS
4.  **Présenter 3 projets prod-ready** sur GitHub pour mon portfolio

**Période** : 26 juin 2026 →  8 Août 2026

📂 Structure du Dépôt

Chaque jour est un dossier autonome pour garantir une progression traçable.
